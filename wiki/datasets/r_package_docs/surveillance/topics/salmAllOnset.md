salmAllOnset: Salmonella cases in Germany 2001-2014 by data of symptoms onset

Salmonella cases in Germany 2001-2014 by data of symptoms onset

Description

     A dataset containing the reported number of cases of Salmonella in
     Germany 2001-2014 aggregated by data of disease onset. The slot
     ‘control’ contains a matrix ‘reportingTriangle$n’ with the
     reporting triangle as described in Salmon et al. (2015).

Usage

     data(salmAllOnset)

Format

     A sts-object

References

     Salmon, M., Schumacher, D., Stark, K., Höhle, M. (2015): Bayesian
     outbreak detection in the presence of reporting delays.
     Biometrical Journal, 57 (6), 1051-1067.
