influMen: Influenza and meningococcal infections in Germany, 2001-2006

Influenza and meningococcal infections in Germany, 2001-2006

Description

     Weekly counts of new influenza and meningococcal infections in
     Germany 2001-2006.

Usage

     data(influMen)

Format

     A ‘disProg’ object containing 312 x 2 observations starting from
     week 1 in 2001 to week 52 in 2006.

Source

     Robert Koch-Institut: SurvStat: <https://survstat.rki.de/>.
     Queried on 25 July 2007.

Examples
Run this code

     data(influMen)
     plot(influMen, as.one=FALSE, same.scale=FALSE)
