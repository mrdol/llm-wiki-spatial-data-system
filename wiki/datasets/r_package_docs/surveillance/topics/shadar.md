shadar: Salmonella Hadar cases in Germany 2001-2006

Salmonella Hadar cases in Germany 2001-2006

Description

     Number of salmonella hadar cases in Germany 2001-2006. An increase
     is seen during 2006.

Usage

     data(shadar)

Format

     A ‘disProg’ object containing 295 x 1 observations starting from
     week 1 in 2001 to week 35 in 2006.

Source

     Robert Koch-Institut: SurvStat: <https://survstat.rki.de/>;
     Queried on September 2006.

     Robert Koch Institut, Epidemiologisches Bulletin 31/2006.

Examples
Run this code

     data(shadar)
     plot(shadar)
