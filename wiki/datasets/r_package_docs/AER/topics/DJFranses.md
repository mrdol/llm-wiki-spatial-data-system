DJFranses: Dow Jones Index Data (Franses)

Dow Jones Index Data (Franses)

Description

     Dow Jones index time series computed at the end of the week where
     week is assumed to run from Thursday to Wednesday.

Usage

     data("DJFranses")

Format

     A weekly univariate time series from 1980(1) to 1994(42).

Source

     Online complements to Franses (1998).

References

     Franses, P.H. (1998). _Time Series Models for Business and
     Economic Forecasting_. Cambridge, UK: Cambridge University Press.

See Also

     ‘Franses1998’

Examples
Run this code

     data("DJFranses")
     plot(DJFranses)
