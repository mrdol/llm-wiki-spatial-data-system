Dataset discovered by the system but not yet validated as a final modeling dataset.

## Definition

A candidate dataset is a dataset-like record found through a warehouse, software source, or scientific paper. It becomes stronger when it has clear access, metadata, variable typology, spatial-temporal structure, license evidence, and paper/modeling evidence.

## Validation Signals

- source family identified
- DOI or stable landing page
- downloadable files or API access
- candidate `Y` and `X` variables
- spatial and temporal support
- file size and format
- license or reuse terms
- linked paper or model equation
