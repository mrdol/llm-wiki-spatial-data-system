# Spatiotemporal Weighted Regression

Spatiotemporal weighted regression (STWR) is a local regression family that
extends geographically weighted regression to space-time observations. It uses
spatio-temporal weights so nearby observations in both space and time influence
local parameter estimates.

## KG Use

The concept should connect papers, formulas and datasets where:

- a local regression is calibrated over space and time;
- the model uses a spatio-temporal kernel or distance-decay function;
- bandwidth selection or cross-validation is reported for space-time local
  prediction.
